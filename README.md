# park-manage
## 公园管理系统(如果对你有帮助，请点右上角star)

## 技术依赖

* vue
* vuex
* vue-router 
* axios
* less
* element-ui
* js-cookie
* mockjs
* echarts
* qs
* php


## 项目预览地址

 www.commit.tk  
(国外服务器数据请求可能有点慢)


## 项目功能

- [x] 登录/注册  
- [x] echarts数据展示  
- [x] 用户管理（用户的增删改查）  
- [x] 游客售票管理  
- [x] 票务管理  
- [x] 管理员申请审核  
- [x] 权限管理（不同权限展示内容不一致）  
- [x] 上传文件  
- [x] 导出excel
