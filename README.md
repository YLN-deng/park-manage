# park-manage
## 公园管理系统(如果对你有帮助，请点右上角star)

## 技术依赖

* vue
* vuex
* vue-router 
* axios
* less
* element-ui
* js-cookie
* mockjs
* echarts
* qs
* php


## 项目预览地址

 www.commit.tk  
(国外服务器数据请求可能有点慢)


## 项目功能
```
登录/注册
echarts数据展示
用户管理（用户的增删改查）
游客售票管理
票务管理
管理员申请审核
权限管理（不同权限展示内容不一致）
上传文件
导出excel
```


## 项目运行
```
yarn install
yarn serve
yarn build
或者
npm install
npm run serve
npm run build
```
